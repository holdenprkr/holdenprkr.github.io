Projects

Crypto Current

Crypto Current is an application used to help assist beginners looking to invest in crypto currency have a place to go to make the right move. Using manipulated data from a third-party API and ChartJS, Crypto Current displays a line graph of popular crypto currencies (displaying Bitcoin by default) with corresponding statistics depending on the time increment selected to view. Using JSON as a local database users can post, edit, and delete their own messages on the public message board as well as follow users they find helpful. The "Following" page displays the usernames of who has been followed along with all their recent messages. Managing state with react hooks was paramount in making everything function fluidly. Lastly, the "Crypto" page allows the user to add cryptos, able to view on the home page, be added to your watch-list as well as favorite or visit recommended crypto exchanges.
